üìò FileOps API Documentation

Fast, simple image processing APIs for developers. Compress, resize, convert and watermark images using HTTP.

Base URL

https://fileops-api.onrender.com

Authentication

All API requests require an API key passed via request headers. 

x-api-key: YOUR_API_KEY

Requests without a valid API key or exceeding daily limits will return an error response.

üìä Check API Usage

Retrieve your current usage, plan limits and remaining requests. 

GET /usage

Example (cURL)

curl https://fileops-api.onrender.com/usage \
  -H "x-api-key: YOUR_API_KEY"

Example response:

{
  "api_key": "fops_live_xxx",
  "plan": "free",
  "used_today": 3,
  "limit": 20,
  "remaining": 17
}

üóúÔ∏è Compress Image

Reduce image size while maintaining visual quality.

POST /compress-image

Example (cURL)

curl -X POST https://fileops-api.onrender.com/compress-image \
  -H "x-api-key: YOUR_API_KEY" \
  -F "file=@image.png"

Returns a compressed image file.

üìê Resize Image

Resize images while preserving aspect ratio.

POST /resize-image?width=400

Example (cURL)

curl -X POST "https://fileops-api.onrender.com/resize-image?width=400" \
  -H "x-api-key: YOUR_API_KEY" \
  -F "file=@image.png"

You may provide width, height, or both.

üîÑ Convert Image Format

Convert images between supported formats.

POST /convert-image?target_format=jpeg

Example (cURL)

curl -X POST "https://fileops-api.onrender.com/convert-image?target_format=jpeg" \
  -H "x-api-key: YOUR_API_KEY" \
  -F "file=@image.png"

Supported formats: jpeg, png, webp

üíß Watermark Image

Add a visible text watermark to images. Ideal for branding and copyright protection. 

POST /watermark-image?text=YourBrand

Example (cURL)

curl -X POST "https://fileops-api.onrender.com/watermark-image?text=FileOps" \
  -H "x-api-key: YOUR_API_KEY" \
  -F "file=@image.png"

Returns the image with a text watermark applied.
