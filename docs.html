FileOps API

FileOps is a simple, fast image processing API for developers. Perform compression, resizing, format conversion and watermarking using secure HTTP endpoints.

Base URL

https://fileops-api.onrender.com

Authentication

All API requests must include your API key in the request headers. 

x-api-key: YOUR_API_KEY

Requests with missing, invalid or exhausted API keys will be rejected.

Check Usage

Retrieve your current plan limits and remaining requests. 

GET /usage
        
curl https://fileops-api.onrender.com/usage \
  -H "x-api-key: YOUR_API_KEY"
        
{
  "plan": "free",
  "used_today": 3,
  "limit": 20,
  "remaining": 17
}

Compress Image

Reduce image file size while preserving quality.

POST /compress-image
        
curl -X POST https://fileops-api.onrender.com/compress-image \
  -H "x-api-key: YOUR_API_KEY" \
  -F "file=@image.png"

Resize Image

Resize images while maintaining aspect ratio.

POST /resize-image?width=400

Convert Image

Convert images to different formats.

POST /convert-image?target_format=jpeg

Supported formats: jpeg, png, webp

Watermark Image

Add a visible text watermark to images.

POST /watermark-image?text=FileOps

Error Responses

{ "error": "Invalid API key" }
        
{ "error": "Daily limit exceeded" }
