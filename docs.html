Getting Started

FileOps API allows you to compress, resize, and convert images using simple HTTP requests.

Base URL

https://fileops-api.onrender.com

Authentication

All requests must include your API key in the header: 

x-api-key: YOUR_API_KEY

If the API key is missing or invalid, the request will be rejected.

Compress Image

POST /compress-image

Example (cURL)

curl -X POST https://fileops-api.onrender.com/compress-image \
  -H "x-api-key: YOUR_API_KEY" \
  -F "file=@image.png"

Returns a compressed image file.

Resize Image

POST /resize-image?width=400

Example (cURL)

curl -X POST "https://fileops-api.onrender.com/resize-image?width=400" \
  -H "x-api-key: YOUR_API_KEY" \
  -F "file=@image.png"

Returns a resized image while maintaining aspect ratio.

Convert Image Format

POST /convert-image?format=jpeg

Example (cURL)

curl -X POST "https://fileops-api.onrender.com/convert-image?format=jpeg" \
  -H "x-api-key: YOUR_API_KEY" \
  -F "file=@image.png"

Supported formats: jpeg, png, webp

Error Responses

Invalid API Key

{
  "error": "Invalid API key"
}

Daily Limit Exceeded

{
  "error": "Daily limit exceeded"
}
